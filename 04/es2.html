Esercizio 4.2

Il metodo di Inline utilizza style nella apertura del tag e vale fino alla chiusura del tag

Prova di Inline

Il metodo di Internal utilizza il tag style nella head del documento HTML e vale per tutti i tag del tipo definito nello style

Il metodo di External utilizza il tag style in un file esterno ed è necessario il tag link per collegarlo al documento HTML, i cambiamenti valgono per tutti i tag dello stesso tipo definti nello style

La proprietà color serve a modificare il colore del testo

La proprietà font-family serve a cambiare il carattere del testo

La proprietà font-size serve a cambiare la dimensione del carattere del testo

La proprietà border è un gruppo di proprietà che serve a specificare le proprietà del bordo

La proprietà padding definisce la distanza del contenuto dal bordo dell'elemento

La proprietà margin definisce il margine
