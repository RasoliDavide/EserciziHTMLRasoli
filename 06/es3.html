Esercizio 6.3

This div element has width: 500px;

Che con width il border va fuori dallo schermo

This div element has max-width: 500px;

Con max-width il border si adatta alla grandezza della finestra
La max-width è più adatta al cellulare

Tip: Drag the browser window to smaller than 500px wide, to see the difference between the two divs!

position: relative;

An element with position: relative; is positioned relative to its normal position:

This div element has position: relative;
This div element has position: fixed;
This div element has position: absolute;

Con position static le proprietà left, right, top e bottom vengono ignorate.
Con relative invece vengono prese in considerazione
Con fixed vengono prese in considerazione, e l'elemento rimane sempre visibile durante lo scroll della pagina
Con absolute vengono prese in considerazione ma scrollando l'elemento non viene visualizzato sempre.
